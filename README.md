## Descrição do projeto:

O projeto simula uma lista de tarefas simplificada onde é possível executar as seguintes operações:

* Cadastrar usuários
* Listar usuários
* Excluir usuários
* Cadastrar tarefas
* Listar tarefas
* Excluir tarefas.

## Stack utilizada:
* Java 17
* Spring boot (3.3.3)
* PostgreSQL (16)
